# 📊 Unemployment Analysis with Python

## 📌 Project Overview

This project analyzes unemployment trends in India using Python. The analysis includes data cleaning, exploratory data analysis (EDA), data visualization, and insights into the impact of COVID-19 on unemployment.

---

## 🎯 Objective

- Analyze unemployment trends across different regions.
- Compare Rural and Urban unemployment.
- Study unemployment changes over time.
- Understand the impact of COVID-19.
- Generate meaningful insights through data visualization.

---

## 📂 Dataset

The dataset contains:

- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📊 Visualizations

- Distribution of Unemployment Rate
- Rural vs Urban Analysis
- Top 10 States with Highest Unemployment
- Monthly Unemployment Trend
- Correlation Heatmap
- COVID-19 Impact Analysis
- Top 10 States by Employment

---

## 💡 Key Insights

- Unemployment varies across different regions.
- Rural and Urban areas show different employment patterns.
- COVID-19 significantly impacted unemployment.
- Data visualization helps identify important economic trends.

---

## ✅ Conclusion

This project demonstrates how Python can be used for real-world data analysis. Through data cleaning, visualization, and interpretation, valuable insights about unemployment trends were obtained.
