# 📊 Unemployment Analysis in India using Python

## About the Project

As part of my CodeAlpha Data Science Internship, I worked on analyzing unemployment data in India using Python. The main goal of this project was to understand unemployment patterns across different states, compare rural and urban areas, and observe how unemployment changed during the COVID-19 pandemic.

Through this project, I practiced data cleaning, data visualization, and exploratory data analysis (EDA) using real-world data.

---

## Objectives

- Understand the unemployment dataset.
- Clean and preprocess the data.
- Explore unemployment trends across India.
- Compare unemployment in Rural and Urban areas.
- Study the impact of COVID-19 on unemployment.
- Create meaningful visualizations and draw insights.

---

## Dataset Features

The dataset includes:

- Region
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area (Rural/Urban)

---

## Tools & Libraries

- Python
- Pandas

- Matplotlib
- Seaborn
- Google Colab

## Visualizations Created

- Distribution of Unemployment Rate
- Rural vs Urban Comparison
- Top 10 States with Highest Unemployment
- Monthly Unemployment Trend
- Correlation Heatmap
- COVID-19 Impact Analysis
- Top 10 States by Employment
